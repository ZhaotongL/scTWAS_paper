# Repository structure

- `R/`  
  Core R functions to run **scTWAS**, **ANTWAS**, and **NATWAS**.

- `toy_example/`  
  A toy example demonstrating how to run **scTWAS**.

- `realdata_code/`  
  R code and scripts for Stage 1 and Stage 2 analyses of scTWAS, ANTWAS, and NATWAS in OneK1K and ROSMAP.
